# 🔔 HAMA信号监控 - 快速入门

## 🚀 3步开始使用

### 步骤1: 部署更新

```bash
# 双击运行
一键部署.bat
```

等待2-3分钟完成部署

### 步骤2: 访问页面

```
http://localhost:8888/hama-monitor
```

### 步骤3: 启动监控

1. 点击 **"启动监控"** 按钮
2. 点击 **"添加涨幅榜"** 按钮
3. 点击确定

完成!监控现在会自动运行,当检测到涨跌信号时会显示在页面中

---

## 📊 页面说明

### 顶部统计
- 📊 监控币种: 当前监控的币种数量
- 🔔 信号总数: 历史信号总数
- ⏱️ 检查间隔: 60秒
- ❄️ 冷却时间: 300秒

### 控制按钮
- **启动监控**: 开始运行监控服务
- **添加涨幅榜**: 批量添加涨幅榜前20名
- **添加币种**: 手动添加单个币种
- **配置**: 修改监控参数
- **刷新**: 刷新页面数据

### 信号历史表格
显示所有检测到的信号:
- 📈 绿色 = 涨信号(上穿MA线)
- 📉 红色 = 跌信号(下穿MA线)

---

## 🔔 信号说明

### 涨信号 (📈)
- **HAMA蜡烛图上穿MA线**
- **可能趋势**: 上涨
- **建议**: 考虑买入

### 跌信号 (📉)
- **HAMA蜡烛图下穿MA线**
- **可能趋势**: 下跌
- **建议**: 考虑卖出

## 📝 常用操作

### 添加币种
1. 点击"添加币种"
2. 输入币种(如 BTCUSDT)
3. 选择现货/合约
4. 点击确定

### 移除币种
1. 在"监控币种"列表中找到币种
2. 点击"移除"按钮

### 清空历史
1. 在"信号历史"标题旁
2. 点击"清空历史"按钮

### 修改配置
1. 点击"配置"按钮
2. 调整检查间隔和冷却时间
3. 点击确定

---

## ⚙️ 推荐配置

### 短线交易
- 检查间隔: 30秒
- 冷却时间: 60秒
- 监控币种: 涨幅榜前20

### 中长线
- 检查间隔: 300秒(5分钟)
- 冷却时间: 600秒(10分钟)
- 监控币种: 精选5-10个
