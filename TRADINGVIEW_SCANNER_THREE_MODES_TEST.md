## 🎨 前端页面使用

### 切换模式
在页面顶部有下拉菜单:
```
┌─────────────────────┐
│ 永续合约      ▼    │
└─────────────────────┘
```

点击下拉菜单选择:
1. 📊 永续合约
2. 🔥 涨幅榜
3. ⭐ 关注列表

### 调整数量
右侧有数字输入框:
```
┌─────┐
│ 50  │  ← 可调整数量
└─────┘
```

- 永续合约: 最多200个
- 涨幅榜/关注列表: 最多100个

### 刷新数据
点击刷新按钮或等待自动刷新(每2分钟)
