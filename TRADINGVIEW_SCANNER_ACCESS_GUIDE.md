# 🌐 TradingView行情页面访问指南

## 📍 页面位置

### 方式1: 通过菜单访问 (推荐)

1. 打开浏览器访问: `http://localhost:8888`
2. 登录系统 (如果需要)
3. 在左侧菜单栏找到 **"TradingView行情"**
4. 点击即可进入页面

**菜单图标**: 📈 (line-chart图标)
**菜单名称**: TradingView行情
**路由路径**: /tradingview-scanner

---

### 方式2: 直接URL访问

在浏览器中直接输入:
```
http://localhost:8888/tradingview-scanner
```

---

## 🎨 页面功能

### 1. 数据类型切换

页面顶部有三个选项:
- 📊 **永续合约** (默认) - 显示78个币安永续合约
- 🔥 **涨幅榜** - 按涨跌幅排序的币种列表
- ⭐ **关注列表** - 20个主流加密货币

### 2. 数量调整

可以通过输入框调整显示数量:
- 永续合约: 最多200个
- 涨幅榜/关注列表: 最多100个

### 3. 刷新按钮

点击刷新按钮手动更新数据,页面每2分钟自动刷新

---

## 📊 页面展示内容

### 统计卡片区域 (顶部)

显示4个统计卡片:
- 📊 **总币种数** - 当前显示的币种总数
- 📈 **平均涨跌幅** - 所有币种的平均涨跌幅 (绿色/红色)
- 📈 **上涨币种** - 涨跌幅>0的币种数量
- 📉 **下跌币种** - 涨跌幅<0的币种数量

### 数据表格区域

显示币种详细信息:
- **排名** - 金银铜牌徽章 (Top3)
- **币种** - 币种符号 (蓝色标签)
- **描述** - 币种全名
- **价格** - 当前价格 (USD)
- **24h涨跌** - 涨跌幅百分比 (带箭头和颜色)
- **24h成交量** - 成交量 (自动单位转换 K/M/B)
- **交易所** - Binance
- **操作** - 跳转TradingView图表按钮

## 🎯 使用场景

### 场景1: 查看永续合约行情

1. 进入页面后默认显示永续合约模式
2. 可以看到78个币种
3. 表格按默认顺序显示
4. 点击"刷新"按钮获取最新数据

### 场景2: 查看涨幅榜

1. 在顶部下拉菜单选择"涨幅榜"
2. 表格自动按涨跌幅降序排列
3. 涨幅最高的在第一位
4. 可以快速发现热门币种

### 场景3: 查看主流币种

1. 在顶部下拉菜单选择"关注列表"
2. 显示BTC、ETH等20个主流币种
3. 适合快速查看市场概况

### 场景4: 查看详细图表

1. 在表格右侧找到"TradingView"按钮
2. 点击按钮跳转到TradingView图表
3. 可以查看更详细的技术分析

## 🔧 技术细节

### API端点

页面调用以下后端API:
- `GET /api/tradingview-scanner/watchlist` - 关注列表
- `GET /api/tradingview-scanner/perpetuals` - 永续合约
- `GET /api/tradingview-scanner/top-gainers` - 涨幅榜

### 数据源

所有数据来自:
- **TradingView Scanner API**
- **Binance永续合约**
- **实时价格和涨跌幅**

### 性能优化

- 批量获取数据 (每批20个)
- 避免API限流
- 缓存优化
- 快速响应
