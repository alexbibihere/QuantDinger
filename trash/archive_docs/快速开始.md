## 🐛 常见问题

### ❌ "Docker未运行"

**解决:**
1. 启动 Docker Desktop
2. 等待完全启动
3. 重新运行部署脚本

### ❌ "端口被占用"

**解决:**
```cmd
netstat -ano | findstr :5000
netstat -ano | findstr :8888
taskkill /PID <进程ID> /F
```

### ❌ "API返回404"

**原因:** 后端需要重启以加载新的涨幅榜路由

**解决:**
```cmd
docker compose restart backend
```

等待30秒后重新访问

### ❌ "容器未启动"

**检查日志:**
```cmd
docker compose logs backend
docker compose logs frontend
```

**重启服务:**
```cmd
docker compose restart
```
